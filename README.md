# Java Student Management System

A simple Java console project demonstrating:
- OOP Concepts
- Classes & Objects
- ArrayList usage
- User input handling
- Clean, interactive menu

### Features
1. Add new students  
2. View all students  
3. Exit system  

### How to Run
1. Save the file as StudentManagement.java  
2. Compile:
   javac StudentManagement.java
3. Run:
   java StudentManagement
